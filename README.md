# Java面向对象初步练习：将公用代码抽取到父类中

> 注：重构指的是在**不改变**原先代码功能的前提下，将原先的代码变得更精炼、可维护性更强。

在提交Pull Request之前，你应当在本地确保所有代码已经编译通过，并且通过了测试(`mvn test`)
